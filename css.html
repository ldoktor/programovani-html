CSS/Kaskádové styly

Co to je CSS

CSS (zkratka pro Cascading Style Sheets) je kód, který určuje vzhled a rozvržení webové stránky. Zatímco v HTML píšete, co na stránce bude (texty, obrázky, tlačítka), v CSS určujete, jak to bude vypadat.

Jednoduchý prompt pro AI

Pro základní seznámení s možnostmi CSS můžeme využít LLM (large language models == veliké jazykové modely) aby nám ukázali, jak se může naše stránka proměnit bez změny obsahu.

Můžeme využít následující "prompt" (=dotaz) jako odrazový můstek a následně již vlasními slovy doplňte, jak by měla vaše stránka vypadat.

Pozor, pokud chcete do promptu přidat novou řádku, musíte kliknout 

shift+enter
. Pomocí 
enter
odešlete dotaz. 


Jsem student a učím se HTML/CSS. Vytvořil jsem následující stránku:

VAŠE STRÁNKA (ctrl+c, ctrl+v)

a rád bych se naučil jak moc ji lze proměnit pomocí css. Pro horní menu a fotky je nutné něco jako:

/* HORNÍ MENU */
#horni-menu {
  position: fixed;
  top: 0;
  width: 100%;
}
#horni-menu ul {
  display: flex;
  list-style: none;
}
body { padding-top: 50px; }

/* GALERIE */
.overlay {
  position: fixed;
  inset: 0;
  display: flex;
}
.overlay.hidden {
  display: none;
}
.overlay img {
  max-width: 100%;
  max-height: 100%;
}
.nav, .close {
  position: absolute;
}
#prev { left: 0; }
#next { right: 0; }
.close { top: 0; right: 0; }
